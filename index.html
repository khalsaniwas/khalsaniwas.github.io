Dholcutz Bhangra Radio

Dholcutz Bhangra Radio is web application for listening online punjabi music. Users can send a request to add a song in queue. The front-end is built using React.js, utilizing Axios for http requests to communicate with the API. Backend is built using AzuraCast, Docker and PHP.

Property Manager

Property Manager is a single page application where users can add their properties to manage. After sign in, user's have access to all of their properties. User's can choose to create new property or edit their existing properties. The back-end was built using Ruby on Rails, and PostgreSQL. The front-end was built using React.js, utilizing Axios for http requests to communicate with the API.

Ask Nicely

"Ask Nicely" is a web application that allows a user to create a survey. They can create 4 response options for that one survey. Other users can then log in, and vote on that survey. The app records how many votes each option has, and can be displayed for all users to see. Users can view all surveys that have been created, and vote on them. A user can also update and delete there own survey, but not others.
